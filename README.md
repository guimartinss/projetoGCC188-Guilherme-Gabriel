O objetivo do projeto é a criação de uma calculadora, feito em 3 etapas.

Etapa 1, lançada como versão 0.1
    - RF01 - Criar a função sucessor, feita por Gabriel
    - RF02 - Criar a função antecessor, feita por Guilherme

Etapa 2, lançada como versão 1.0
    - RF03 - Criar a função adição, feita por Gabriel
    - RF04 - Criar a função subtração, feita por Guilherme

Etapa 3, lançada como versão 2.0
    - RF05 - Criar a função multiplicação, feita por Gabriel
    - RF06 - Criar a função divisão, feita por Guilherme
    - RF07 - Criar a função resto, feita por Guilherme
    - Fix01 - RF04 deve retornar 0 se o resultado for negativo, feita por Guilherme
    - Fix02 - Possibilidade de somar entre 2 e 4 números, feita por Gabriel
